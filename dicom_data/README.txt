================================================================================
DICOM Data Directory
================================================================================

English:
--------
This directory should contain your DICOM files for conversion.

Please place your DICOM files (e.g., DCM00000, DCM00001, etc.) in this
directory before running the converter.

Example structure:
  dicom_data/
    ├── DCM00000
    ├── DCM00001
    ├── DCM00002
    └── ...

To convert:
  python3 dicom_converter.py


中文说明：
--------
此目录应包含要转换的DICOM文件。

请在运行转换器之前将DICOM文件（例如DCM00000、DCM00001等）放入此目录。

目录结构示例：
  dicom_data/
    ├── DCM00000
    ├── DCM00001
    ├── DCM00002
    └── ...

运行转换：
